# CloseExp

いつのまにかたくさん開いてしまった、ブラウザやエクスプローラーウインドウなどを一括で閉じるフリーソフトです。

気づいたらブラウザのウインドウがいっぱいになって重い・・・ということはありませんか？
そんなときに一番上のウインドウを残して他のウインドウを簡単に閉じるソフトです。

1999年頃に作成したものですが、いまだに引き合いがあるので最低限現在の環境でも使えるようにして公開します。
コードは古い上に汚いですがお許しください。


# Requirement

*Windows 10/11あたりで動作します。プロジェクトはVisualStudio2019ですが、何のライブラリも不使用です。
*コード的にはWindows95あたりでも動くと思いますが、バイナリはXP以前では動かないと思います。

ブラウザの仕様がかわると検出できなくなったり、余計なウインドウを捕捉してしまったりするので、必要に応じて修正する予定です。
